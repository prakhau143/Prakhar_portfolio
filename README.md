## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/prakhau143/Prakhar-mittal-portfolio.git
   cd Prakhar-mittal-portfolio-in-3d
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm start
   ```
